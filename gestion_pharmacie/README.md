# Système de Gestion de Distribution  des médicaments

![Page d'accueil](screenshots/accueil.PNG)

Application web Django pour gérer les opérations d'un distributeur  des médicaments

## Description

Système complet de gestion permettant de gérer les clients (pharmacies,hospitaux,cliniques),
fournisseurs (laboratoires), stock de médicaments et commandes avec mise à jour automatique du stock.

## Fonctionnalités principales

- Authentification sécurisée (connexion/inscription)
- Gestion CRUD complète : Clients, Fournisseurs, Médicaments, Commandes
- Diminution automatique du stock lors des commandes
- Alertes "Stock insuffisant" et "Stock vide"
- Interface responsive

## Technologies

Python 3.12 | Django 5.2 | SQLite | HTML5/CSS3
